Legal High

2017-12-29

古美门在第二季的出场形象显然是个间歇性精神病，大家都以为这是legal high是走向无节操、无底线、爆笑型喜剧的节奏。甚至古美门自己也是这样认为，作为一个天才，从来都是所向披靡，他的生活该是欢乐的、不羁的。让古美门这么一路赢下去，是最讨巧的做法，更加浮夸的表现会带来更多的喜剧效果。可是编剧没有，他非让古美门在眼看要赢得官司的时候惨败。大家以为，古美门的失败，是因为当事人的背叛，是时运不济，他终究可以大胜而归。到最后，这将成为一个老套的天才的故事，源源不断地讲述一个天才获胜故事。直到第六集，才发现其实编剧用心良苦。
当事人当庭翻供，古美门甚至没有任何反击的机会，就这样败了。他的不败神话破灭了，他崩溃了，因为他想不明白，千算万算，为什么最后是他所辩护的当事人背叛他。他以为是命运背叛了他，他选择了逃避，远离背叛了他的世界。黛的两巴掌激起了他的斗志，他重新振作了起来，决心向背叛他的世界挑战，继续赢下去，反败为胜成为他的目标。可是，他真的觉醒了吗？古美门还是古美门，只不过是有了一次惨败记录仍不肯服输的古美门。为了证明自己不会输，也不能输，他要更加不择手段地获胜，一次次将对手打入地狱。他对敌人的态度愈发的敌对和夸张，他一次次涂抹羽生的照片，就是想告诉所有人，我没有输。他过分的表现不过是戏谑背叛过他的那个世界，想要宣称自己对这个世界的藐视。他拼命地去获胜，却回避反思自己的问题。
第二集里的那个天才鲇川光，其实和古美门是一样的人。他热爱自己的工作，为了达到目的不择手段。他们都能做出最理性的选择，用过人的智慧获得别人无法企及的成功。但是，他们都忘记了去关注是不是有人成为了他们胜利的牺牲品。繁华过后是落寞，鲇川光入狱。而出狱的他依然不改狂妄的本性，居然独自挑战古美门。古美门以为鲇川光是过于狂妄，喜欢上了在法庭上战斗来满足自己的胜利欲望，所以迎战鲇川光，告诉他法律不是儿戏，他的战场只有他能获胜。直到最后，他发现了鲇川光真正的目的，才意识到这个人最了不起的地方，甚至不由地感慨：幸亏鲇川光无意于成为他的敌人。这一次，古美门似乎意识到了，在胜利之外，还有一些东西更值得珍惜。可是，他又懂得了多少？为什么三木总嚷嚷着要把他打入地狱，难道只是因为一只仓鼠么？或许，古美门和鲇川光一样，需要历经地狱，才能重生。
第三集看上去显得无厘头，却充满了编剧对古美门的期待。这一次，古美门当然还要赢。管他什么人情常伦，只要是在他古美门严密的逻辑下说得通的，就能将一切逆转，获得最后的胜利。然而，那个女人确实很可怜，古美门也是人，难免有怜悯之心，所以在她离开时给了她鼓励。按照从前古美门的习惯，他是断然不会这么做的。可能是之前经历的一切，给了他一点触动，让他忍不住有了情感的流露。可是，他还只是站在一个旁观者的角度，仅仅给人施舍一点怜悯。而这时的羽生，他的理想和信念被击败了，当事人的痛苦他感同身受，所以编剧自然将橄榄枝抛给了他。
第四集无疑是编剧对依然深陷胜利中的古美门的惩罚，也是对羽生的奖励。古美门在法庭获胜了，羽生却用另一种形式获胜了，这一次，他真的实现了他的理想。而古美门似乎意识到了这一点，但却又鄙视羽生的不现实。现实确实是必须面对的最严峻的问题，法官都身不由己，何况小小的律师。古美门的逻辑再一次胜利了，他和法官的对话，看似是讨嫌，其实是向她宣告，我的方法在现实里是有效的，我是胜利的。可是，法官却说，你也是不自由的。编剧借法官的口，对古美门表达了最大的同情。古美门不是不知道这个事实，但他宁可固执地守着胜利的欲望，也不要别人的同情。又是一个无厘头的结局，多么痛的领悟。只是希望，他真的领悟。
第五集在狗老爹所有权的争斗中，编剧客观地对古美门的逻辑作出评判。从法律的角度，古美门没有错，他是一个尽职尽责的法律守护者；从现实的角度，古美门崇尚的现实至上的原则也没有错，优胜劣汰，适者生存，虽然来得残酷，但必须经历，必须接受。这一次，古美门赢得有理有据，赢得正当。只是，在法律之外，现实之外，还有东西是古美门没能发掘的。就是人的情感、人的内心的力量是巨大的，斗争除了针锋相对，还可以有另一种结局。古美门坚持他所认为对的东西，这是他成功的精髓，也是他的软肋。而真正能触及这个软肋，给他致命打击的人就是黛。
一妻多夫看似荒诞，却饱含了编剧的良苦用心。因为这是古美门第一次真正看到，在他的逻辑失效时，能帮他获胜的居然是他平时嗤之以鼻的情感。如果三个丈夫对妻子真的没有真爱了，仅凭兰丸的几句话是不会让他们回心转意的。一妻虽然有三夫，但她都付出了真爱，直到他们纷纷离去，她依然希望他们都能幸福。这样的爱，理应获得真诚的回报。这一次胜利后，古美门一定在暗暗地感慨吧。直到小雪的几句话，突然给古美门一次重击。坐在家中，古美门一定想了很多。他有的获胜技能，黛已经有了，而黛有的对人性的关怀，对情感的发掘，他却没有。获胜的欲望几乎要将他吞没了，当他警觉到这一点时，他意识到不能让黛也被吞没。而黛的感情显然已经影响了她，渐渐无法去维护自己那些珍贵的品质。与其说黛在古美门的庇护下不愿离开，更不如说古美门希望黛能弥补自己的不足而不愿放手。为了保护黛身上仅存的希望，古美门必须赶走她。终于走了，清净了，古美门是该好好想想自己的问题了。他所坚持的，所相信的，究竟还能带他走多久，还能获得多少胜利；黛与他的对抗，会帮助他警醒么，带他成长么？无论如何，只有出发了，才能看到最后的答案。
其实，从古美门一根手指的手势就足以看出些端倪。只有在他自信满满、占尽上风的时候，他才会坚定地举起那根手指。他的手指是冲向他面对的世界，告诉所有人法律是唯一的逻辑，胜利是唯一的目标，谁都不可以动摇。随着他的成长，可能有一天他会好好低头看看自己的这根手指，想想什么才是唯一，什么才该永远坚持。
我真的很希望编剧，在最后的时候，无论古美门的成败，都能将那根手指指向黛，告诉她，你才是唯一。
·····················一点也不华丽的分割线·····································
第七集，古美门还是那个古美门，毒舌、自负、猥琐。只是这一次，他又添加了一项技能，死皮赖脸追在黛屁股后面喊打。黛虽然走了，作为黛的老师的古美门依然尽职尽责，鹦鹉案、暴露狂案，不过是告诉黛，所谓的love&peace不过是失败者的自我安慰，真正的胜利只有通过不断的战斗才能获得。正如小雪说的，古美门心里很明白，只有黛成为真正的对手，才有可能打败他，让他觉醒。只有无惧于失败，才能无往不胜。要觉醒，古美门要打败那个极端畏惧失败的自己，而黛显然就是那个年轻的更有希望的古美门，在他的调教下，黛逐渐变得和古美门一样睿智、强硬，不同的是她少了古美门的世故，多了无畏失败的韧劲，只有这样的古美门才能战胜现在的古美门。不得不说，小雪之所以能成为“世纪恶女”也是有一定水平的，几句话就点透了古美门行动的始终。但是，如果故事只停留在这里，那似乎简单得有些可笑。
其实，真正的重点是漫画导演案。不难看出，漫画导演和员工的关系和古美门与黛有许多相似，一个天才式的人物对有希望的学生采用魔鬼般的训练，背后的却是用心良苦。无论如何，这份良苦用心古美门最能理解和体会，他想看看黛会如何反驳这份用心，这场战斗的输赢赌上的是人心。你见过这么严肃的古美门么？这一次，他不愿输，特别是输给黛。黛对自己的老师是尊敬的，她必然不会去否定老师的用心，而她的出牌也是人心，将心比心，老师的良苦用心要被肯定，学生的内心是否也该被关注？这一次，黛似乎要赢了。可是，故事还没有结束，漫画导演的一席话，震惊了所有人，也惭愧了古美门。
没有人是天才，个人的才能是靠自己挖掘的，登上山顶，望望身后已看不见的芸芸众生，回过头来，还需要继续向上。生活本不是闲情逸致，它本就是丑陋的，这个世界不会因为谁的离开就停止送给你恶意。而我，要教给你的，就是去战斗，即使我不在了，你也能独自无畏地去战斗。
·····················不情不愿的分割线···········································
第八集，胜利是一场梦魇。她看似光鲜的外表下，也很有可怕的一面。羽生变了，显然他的行为已经引起了众怒。让所有人幸福的想法没有错，他的正义感也没有错，只是他斗不过人性，斗不过现实。在一次次的失败中，他囤积了太多的失望，渴望胜利的情绪渐渐占据了他的内心。为了胜利，他要成为道德的君王，要不择手段让现实拜倒在道德的脚下。羽生的失败，可叹，也可悲。在胜利这场梦魇中，他无疑成为了彻底的牺牲品。古美门胜利了，他守护了村民的意愿，维护了他们的欲望。可是他做对了吗？回头望望优美的森林，不由得想这可能是最后一次看到了，而正是他一手促成了这美景的消失。内心的道德感和法律的正义在古美门心中交锋，他的胜利真的有意义吗？这场胜利，带来的很可能是森林的毁掉，带来的是人们无尽的遗憾，胜利之痛谁又能承担？
别府法官其实和古美门是一类人，她同样行事另类，同样遭受别人异样的眼光，但却始终坚信无论在什么地方，什么凄凉的境地，自己都最适合做一位法官。黛问古美门，她会成为一位名法官吧。古美门却说，不会。因为古美门明白，别府法官要做的是一个纯粹的、守护法律的法官，这与声名无关，而是她的命运和终身的追求。对于古美门，也是一样。不同的是，他还没能那样勇敢，那样无畏。在胜利魔棒的指挥下，他还能按照自己的意愿前行吗，他能够选择那条最适合的人生道路吗？
古美门似乎已经开始重新审视自己所相信的，思考究竟什么该是真正去坚持的。胜利就是正义吗？法律的正义和现实的规则，他在好好坚守吗？为杀人犯辩护，他要去坚持什么，要守护什么，胜利不会告诉他答案。
·································迟到的分割线····································
第九集，人心之重。法律不是完美的，不能囊括社会的方方面面，在道德和法律的灰色地带，法律能够守护的只是底线，而道德的审判却往往被人为操控。在所谓声势浩大的卫道者的讨伐中，古美门的压力不言而喻。为杀人犯辩护，夹在法律制裁和道德审判中间，失败似乎已成定局，他害怕了，经历过惨痛的失败，他不知道自己还能否再次历经打击站起来。总是失败的小怪兽在被奥特曼一次次打倒之后，还能卷土重来，他却办不到。他见到过人心的可贵，知道人心之重。不同的是，这一次，人心的恶魔聚集了起来，天平已经倒向另一边，他已无力应对，只能怯懦地躲在家里，逃避一切。
黛却不肯放弃，即使知道自己势单力薄，也要一搏。她希望自己的受伤能够扭转一些人心，虽然这效果微乎其微。黛作为古美门的学生，始终坚守着他教给她的信念，在老师都放弃的时候依然无悔，愿意伸手去拯救摔落在阴沟被众人攻击的可怜人。人生而平等，黛这样做，表达的不仅是对法律的敬意，还是对善良人心的尊重。对于古美门来说，这样的人心已经足够压平他心中的天平，给他重新站起来的理由。他要重新去战斗，而人心也是他最后的筹码。
人心的恶魔挑战的是法律的权威，对证据的漠视，对黛的伤害，都源自于一群卫道者的人心。所谓民意，在赤裸裸地攻击法律的公正，作为最高法院，自然不能放任。古美门正是抓住了这一点，用法律的公正守住了最后的底线。人心的恶魔肆意妄为，最终也瓦解了自己。人心的善意即使弱小，也可能改变世界。
·······························最后的分割线······································
请爱这丑陋的世界，请爱真实的自己。
真假对错，在这个世界本没有确切的答案，这世界唯一的真相就是它本没有人想象的那么美好，现实总是露出丑陋的一面。无论你怎样愤怒，怎样抗争，都不可能改变这一现实。因为，是人组成了这个社会，而人心是复杂的，时好时坏的人心在不断变换的时间与空间中，势必时而美丽动人，时而丑陋不堪。美好的东西让人愉悦，却也让人易于忘却，丑恶的东西让人痛苦，却让人刻骨铭心。不难理解，经历了一些风雨的人们，总是让心中痛苦的记忆驱使自己去批判世界的丑陋，让自己去痛恨世界，厌弃他人。可是，当人抛弃了丑陋的世界，就连同与它相生的美好也一并丢弃了，终日生活在愤怒与不满中，这不是幸福，也不是这世界该有的样子。请爱这世界，享受它的美好，也接受它的丑陋。
好和坏的定义，本来就在不停摇摆，生活本来就有两面，根本不存在绝对正反的人生。强烈地厌恶所谓的“坏”，厌恶所有“坏”的人，要惩戒一切做“坏”事的人，最终厌恶的会是自己，惩罚的也是自己，因为，这世界本无完人。死生一瞬，人生本不该充满厌恶与仇恨。请爱真实的自己，用爱的态度面对“好”的一面，认清接纳“坏”的一面，无论胜利或失败，都用积极的态度去面对自己的人生。

人心无论美丑，都该得到公正的评判，世界无论丑陋与否，都该守住良知的底线，这是法律的精神，也是人心的准则。法律能够给我们规则，而人心能教会我们爱。
法律为正，人心为重。

python中as用法实例分析

2017-09-03

本文实例讲述了python中as用法。分享给大家供大家参考。具体分析如下：
import some # some 为一个模组
如果想要改变被导入模组在当前模组中的名称，而不是sys.modules中的名称。可以使用import as，例如：
import some as other
print(other.name)
和
import some
other = some
del some
print(other.name)
一样。
